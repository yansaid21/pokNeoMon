Bases de Datos II
Corte 3 - Tarea Pokemon + Mongo + Neo4j

Instrucciones para incorporar su implementación:
================================================

1. Reemplace los archivos battle-crud.js y pokemon-crud.js con sus implementaciones.
2. Ejecute 'npm install' para instalar las dependencias de la aplicación
3. Ejecute 'npm install ...' con aquellos modulos que necesite para su implementación (drivers de las bases de datos)
4. Adicione a la función queryPokemon de pokemon-crud.js la opción de especificar 'id' como criterio de búsqueda.
5. Ejecute 'node server.js' sobre el directorio de la aplicación.
6. Abra con su navegador el archivo 'index.html' y compruebe el funcionamiento de la aplicación.
